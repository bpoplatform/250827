## 🔍 조회 테스트 시나리오

### **시나리오 1: 법인명으로 조회**
1. 메인 페이지(`index.html`)를 열기
2. **법인명** 필드에 입력:
   - `삼성전자` 또는 `삼성전자 주식회사`
   - `엘지전자` 또는 `엘지전자 주식회사`
   - `현대자동차`
   - `네이버`
   - `카카오`
3. **조회** 버튼 클릭
4. ✅ **예상 결과**: 해당 법인 정보와 사업연도 데이터가 로드됨

### **시나리오 2: 법인등록번호로 조회**
1. **법인등록번호** 필드에 입력:
   - `123-45-67890` (삼성전자)
   - `234-56-78901` (엘지전자)
   - `345-67-89012` (현대자동차)
   - `456-78-90123` (네이버)
   - `567-89-01234` (카카오)
2. **조회** 버튼 클릭
3. ✅ **예상 결과**: 해당 법인 정보와 사업연도 데이터가 로드됨

### **시나리오 3: 부분 일치 조회**
1. **법인명**에 일부만 입력:
   - `삼성` (삼성전자 조회됨)
   - `엘지` (엘지전자 조회됨)
   - `현대` (현대자동차 조회됨)
2. **조회** 버튼 클릭
3. ✅ **예상 결과**: 일치하는 법인이 조회됨

### **시나리오 4: 존재하지 않는 데이터 조회**
1. **법인명**에 입력: `존재하지않는회사`
2. **조회** 버튼 클릭
3. ✅ **예상 결과**: "조회된 법인 정보가 없습니다." 메시지 표시

### **시나리오 5: 빈 값으로 조회**
1. 모든 필드를 비우고 **조회** 버튼 클릭
2. ✅ **예상 결과**: "법인명 또는 법인등록번호를 입력해주세요." 메시지 표시

## 🎮 추가 기능 테스트

### **수정 및 저장 테스트**
1. 샘플 데이터 중 하나를 조회
2. 사업연도 정보 수정 (예: 비고란 변경)
3. **저장** 버튼 클릭
4. 다시 **신규** → 같은 법인 **조회**하여 변경사항 확인

### **엑셀 내보내기 테스트**
1. **엑셀저장** 버튼 클릭
2. ✅ **예상 결과**: 모든 법인과 사업연도 정보가 CSV 파일로 다운로드

### **삭제 기능 테스트**
1. 샘플 데이터 중 하나를 조회
2. **삭제** 버튼 클릭
3. 확인 대화상자에서 **확인** 클릭
4. 다시 조회 시도하여 삭제 확인

## 🔧 유효성 검증 테스트

### **중복 등록 방지 테스트**
1. 기존 법인등록번호(`123-45-67890`)로 **신규** 등록 시도
2. ✅ **예상 결과**: "이미 등록된 법인등록번호입니다." 오류 메시지

### **사업연도 겹침 테스트**
1. 삼성전자 조회 후 새 사업연도 추가
2. 기존 기간과 겹치는 기간(`20220601~20230531`) 입력
3. **저장** 시도
4. ✅ **예상 결과**: 기간 겹침 오류 메시지
